# 🎮 Video Game Info DB 🎮

Web application used to search for and display filterable information on video games using the RAWG API.  

---

## ⚙️ Technologies Used ⚙️ 

- HTML5
- CSS3
- JavaScript
- jQuery
- Bootstrap
- RAWG API
- Netlify (for deployment)
